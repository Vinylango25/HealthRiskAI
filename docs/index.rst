Overview
--------

HealthRisk AI is an AI-driven health risk assessment platform integrating:

- **Clinical NLP** — Bio_ClinicalBERT for discharge note analysis and NER
- **Graph Neural Networks** — Heterogeneous GNN for patient–disease–drug graphs
- **Survival Analysis** — Cox PH, DeepSurv, Dynamic DeepHit
- **Tabular Ensembles** — XGBoost / LightGBM / CatBoost stacking
- **Financial Analytics** — Actuarial, hospital credit, pharma pipeline
- **HealthRisk Lab** — Monte Carlo simulation engine
